# minimal.

A music player in a single HTML file. No framework. No build step.

**[Live →](https://upadhyaysiddharth220208-png.github.io/minimal/)**

---

Load a folder. It remembers it.

Albums, artists, tracks, playlists, favourites, history, queue, EQ, crossfade, timed lyrics, sleep timer. Installs as a PWA.

---

**Built on**

File System Access API · Web Audio API · IndexedDB · Media Session API · LRC parser · requestAnimationFrame
